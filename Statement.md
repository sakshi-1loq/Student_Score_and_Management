Student Score and Management System – statement.md

Problem Statement

Managing student records manually is time-consuming and error-prone. Teachers often struggle to calculate total marks, assign grades, and retrieve specific student information quickly. A simple and automated system is needed to store student details, calculate scores, and display results efficiently.

Scope of the Project

This project provides a basic management system that allows users to add students, store their marks, compute totals and grades automatically, and retrieve or display records as needed.
The scope covers:

Data entry of students

Marks management

Automated calculations

Searching and viewing records
Simple and user-friendly menu-based interface


Target Users

School and college students learning Python

Teachers who need a small tool for managing scores

Beginners who want to understand file handling and logic building

Anyone who wants a lightweight student management program


High-Level Features

Add and store student details
Enter marks for multiple subjects

Automatic total and grade calculation

View all student records

Search records by name

Easy-to-use console-based interface
