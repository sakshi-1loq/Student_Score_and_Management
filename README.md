Student Score and Management System

Overview of the Project

This project is a simple Python-based system that allows users to add student details, enter their marks, calculate total and grade automatically, and display the results in a clean format.
It is designed for beginners to understand file handling, functions, and basic logic building.

Features

Add new student details
Enter marks for multiple subjects

Automatic grade calculation

View student data

Search student by name

Update or delete records (if you want, I can add in code)

Simple and clean output


Technologies / Tools Used

Python (Core Python)

File Handling / Dictionary

VS Code or PyCharm or Google Colab

GitHub for repository
Steps to Install & Run the Project

1. Install Python (version 3.8 or above)


2. Download or clone the GitHub repository


3. Open the .py file in your editor


4. Run the program using:

python main.py


5. Follow on-screen instructions
